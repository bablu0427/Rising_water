Project workflow

From dataset to cloud deployment

Rising Waters guides learners through a complete machine learning pipeline for disaster preparedness.

1

Environment Setup

Install Python, Anaconda or Jupyter Notebook, Flask, NumPy, Pandas, Scikit-learn, Seaborn, and XGBoost.

2

Dataset Collection

Use open-source flood prediction datasets with rainfall, cloud visibility, and seasonal weather features.

3

Analysis and Preprocessing

Study distributions, heat maps, missing values, outliers, feature scaling, and train-test splits.

4

Model Building

Train and compare Decision Tree, Random Forest, KNN, and XGBoost using accuracy and classification reports.

5

Flask Deployment

Save the best model as floods.save, connect it to Flask, and deploy it on IBM Cloud.
